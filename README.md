# Arena Battle

`Arena Battle`은 Python과 Pygame으로 만든 자동 전투 아레나 게임입니다.  
여러 캐릭터가 경기장 안에서 자동으로 움직이며 각자의 고유 스킬을 사용해 마지막까지 살아남는 캐릭터가 승리합니다.

---

## 1. 게임 실행에 필요한 준비물

이 게임을 실행하려면 아래 프로그램이 필요합니다.

### 필수 프로그램

- Python 3.10 이상 권장
- pip
- pygame

Python이 설치되어 있지 않다면 먼저 Python 공식 설치 프로그램을 이용해 설치해야 합니다.

설치할 때 Windows에서는 반드시 아래 옵션을 체크하는 것을 추천합니다.

```text
Add python.exe to PATH
```

이 옵션을 체크하지 않으면 터미널에서 `python` 명령어가 작동하지 않을 수 있습니다.

---

## 2. 파일 구성

게임 폴더 안에는 최소한 아래 파일이 있어야 합니다.

```text
ArenaBattle/
├─ main_with_info_readme.py
├─ README.md
├─ blue.jpg
├─ red.jpg
├─ sungmin.jpg
├─ jungwoo.jpg
├─ suin.jpg
├─ sunghyun.jpg
└─ gski.jpg
```

이미지 파일은 없어도 게임 자체는 실행됩니다.  
다만 이미지 파일이 없으면 캐릭터 사진 대신 색깔 원으로 표시됩니다.

### 캐릭터 이미지 파일명

| 캐릭터 | 필요한 이미지 파일 |
|---|---|
| 민재 | `minjae.jpg` |
| 재민 | `jaemin.jpg` |
| 성민 | `sungmin.jpg` |
| 정우 | `jungwoo.png` |
| 수인 | `suin.jpg` |
| 이인록2 | `sunghyun.jpg` |
| 경식 | `gski.jpg` |

이미지를 적용하려면 `.py` 파일과 같은 폴더에 위 파일명 그대로 넣으면 됩니다.

## 3. Windows에서 실행하는 방법

### 3-1. 게임 폴더 열기

예를 들어 게임 파일이 바탕화면의 `ArenaBattle` 폴더 안에 있다면, 파일 탐색기에서 해당 폴더를 열어 주세요.

폴더 빈 공간에서 우클릭한 뒤 아래 중 하나를 선택합니다.

```text
터미널에서 열기
```

또는 PowerShell을 직접 열고 아래처럼 이동합니다.

```powershell
cd "C:\Users\사용자이름\Desktop\ArenaBattle"
```

경로에 한글이나 공백이 있으면 따옴표를 붙이는 것이 안전합니다.

### 3-2. pygame 설치

터미널에서 아래 명령어를 입력합니다.

```powershell
python -m pip install pygame
```

만약 `python` 명령어가 작동하지 않으면 아래 명령어를 사용해 보세요.

```powershell
py -m pip install pygame
```

### 3-3. 게임 실행

```powershell
python main.py
```

또는

```powershell
py main_.py
```

## 6. 가상환경으로 실행하는 방법

여러 Python 프로젝트를 관리한다면 가상환경을 사용하는 것이 좋습니다.

### Windows

```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install pygame
python main_with_info_readme.py
```

### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install pygame
python3 main_with_info_readme.py
```

가상환경을 종료하려면 아래 명령어를 입력합니다.

```bash
deactivate
```

## 8. 게임 모드 설명

### 1대1

두 명의 캐릭터가 작은 경기장에서 싸웁니다.  
가장 기본적인 모드입니다.

### 1대1대1

세 명의 캐릭터가 확장된 경기장에서 싸웁니다.  
캐릭터가 하나 늘어나기 때문에 난전이 더 자주 발생합니다.

### 1대1대1대1

네 명의 캐릭터가 최대 경기장에서 싸웁니다.  
스킬이 동시에 많이 발동하므로 가장 전투가 복잡합니다.

### 아수라장

현재 등록된 모든 캐릭터가 한 번에 투입됩니다.  
캐릭터 선택창을 거치지 않고 바로 전투가 시작됩니다.  
아수라장 모드에서는 전용 초대형 경기장을 사용합니다.

## 9. 캐릭터 소개

### 민재

민재는 레이저와 회피형 스킬을 사용하는 캐릭터입니다.

보유 스킬:

- 고정 레이저: 조준 후 직선 레이저를 발사해 큰 피해를 줍니다.
- 재간둥이: 일정 시간 피해를 받지 않는 회피 상태가 됩니다.
- 착지 충격: 재간둥이 종료 시 주변 적에게 약한 피해를 줍니다.

### 재민

재민은 오오라와 각성 능력을 사용하는 캐릭터입니다.

보유 스킬:

- 오오라: 주변으로 퍼지는 원형 오오라를 만들어 피해를 줍니다.
- 각성재민: 일정 시간이 지나면 잠시 각성하여 오오라 쿨타임이 짧아집니다.
- 맵 오오라: 경기장 전체에 강한 광역 피해를 줍니다.

### 성민

성민은 검과 참격을 사용하는 근접 전투형 캐릭터입니다.

보유 스킬:

- 조준 참격: 상대를 조준한 뒤 빠른 검격을 날립니다.
- 달빛가르기: 맵 전체를 가르는 긴 검기를 발사합니다.
- 검 장착: 이동 방향으로 검을 들고 움직입니다.

### 정우

정우는 소 돌진을 사용하는 캐릭터입니다.

보유 스킬:

- 소 돌진: 오른쪽에서 왼쪽으로 소가 튀어나와 적을 들이받습니다.
- 소떼 돌진: 여러 마리의 소가 맵 전체를 가로지릅니다.
- 정우 보호: 자신이 부른 소 공격에는 피해를 받지 않습니다.

### 수인

수인은 까마귀와 흡혈형 악마화를 사용하는 캐릭터입니다.

보유 스킬:

- 까마귀 손아귀: 적 위치에 장판을 예고한 뒤 피해와 끌어당김을 줍니다.
- 악마화: 주변 적에게 지속 피해를 주고 수인이 회복합니다.
- 악마화 폭발: 변신 종료 시 주변 적에게 폭발 피해를 줍니다.

### 이인록2

이인록2는 배틀그라운드 느낌의 총기와 회복 아이템을 사용하는 캐릭터입니다.

보유 스킬:

- 총 발사: 일정 주기마다 가까운 적을 향해 총알을 발사합니다.
- 회복 아이템: 에너지드링크, 붕대, 구급상자 중 하나를 확률로 사용합니다.
- 회복량 순서: 에너지드링크 < 붕대 < 구급상자 순서로 회복량이 큽니다.

### 경식

경식은 해물찜을 던져 장판 피해를 주는 캐릭터입니다.

보유 스킬:

- 해물찜 투척: 적에게 해물찜을 던져 착지 지점에 열기 장판을 남깁니다.
- 열기 장판: 원형 범위 안의 적에게 지속 피해를 줍니다.
- 해물찜 폭격: 맵 전체 랜덤 위치에 해물찜을 여러 개 떨어뜨립니다.

## 11. 자주 발생하는 오류 해결

### 11-1. `ModuleNotFoundError: No module named 'pygame'`

pygame이 설치되지 않은 상태입니다.

아래 명령어를 실행하세요.

```bash
python -m pip install pygame
```

Windows에서 안 되면 아래 명령어를 사용하세요.

```bash
py -m pip install pygame
```

macOS 또는 Linux에서는 아래 명령어를 사용하세요.

```bash
python3 -m pip install pygame
```

### 11-2. `python` 명령어가 인식되지 않음

Python이 PATH에 등록되지 않았을 가능성이 큽니다.

해결 방법:

1. Python을 다시 설치합니다.
2. 설치 화면에서 `Add python.exe to PATH`를 체크합니다.
3. 터미널을 새로 열고 다시 실행합니다.

Windows에서는 아래 명령어가 대신 작동할 수도 있습니다.

```powershell
py main_with_info_readme.py
```

### 11-3. 이미지가 안 나옴

이미지 파일이 없거나 파일명이 다를 가능성이 큽니다.

확인할 것:

- `.py` 파일과 이미지 파일이 같은 폴더에 있는가?
- 파일명이 정확한가?
- 확장자가 `.jpg`인가?

예를 들어 수인 이미지는 반드시 아래 이름이어야 합니다.

```text
suin.jpg
```

### 11-4. 한글이 네모로 표시됨

코드에서는 `malgungothic` 폰트를 사용합니다.  
Windows에서는 대부분 정상 출력됩니다.

macOS 또는 Linux에서 한글이 깨진다면 시스템에 한글 폰트가 없을 수 있습니다.  
이 경우 코드의 폰트 부분을 시스템에 설치된 한글 폰트로 바꿔야 합니다.

예시:

```python
font = pygame.font.SysFont("malgungothic", 38)
```

이 부분을 다른 한글 폰트 이름으로 변경하면 됩니다.

### 11-5. 게임 창이 바로 꺼짐

터미널에서 실행하지 않고 파일을 더블클릭하면 오류 메시지를 확인하기 어렵습니다.

반드시 터미널에서 아래처럼 실행하세요.

```bash
python main_with_info_readme.py
```

그래야 오류 메시지를 확인하고 문제를 고칠 수 있습니다.
